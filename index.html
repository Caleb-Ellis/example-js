Example JS

This is a small script written in javascript to convert a link to a pattern example with a code block.

Usage

  • Using href to link to the components example source page
  • Class js-example to indicate to the script to use this link as an example
  • Data attrbute data-height to set the height of the iframed example. The width will be set to 100% to full its container

Example

<a href="/pattern/card.html" class="js-example" data-height="170px">Example link</a>
